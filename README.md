# Stankin Schedule Editor

The project is designed to create a weekly schedule in the form of pdf-files.

## Requirements

- Python 3.X
- PyQt5 (```
        pip install PyQT5
        ```)
- PyFPDF (```
        pip install fpdf
        ```)
